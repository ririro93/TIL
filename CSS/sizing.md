# Responsive, relative size
## em
상위 1배면 1em 2배면 2em
- used for media queries

<br>

## rem
root html 기본 폰트 사이즈가 16px 이면 <br>
2rem = 32px
3rem = 48px
- used for sizing and spacing

<br>

## vw, vh

ex). 50vw면 가로 넓이이의 반 크기
